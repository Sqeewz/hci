==============================================================================
                    คู่มือการเล่นเกม HCI: CODE ESCAPE RUNNER
==============================================================================

[1. ภาพรวมของเกม (GAME OVERVIEW)]
------------------------------------------------------------------------------
HCI: Code Escape คือเกม Action Logic Puzzle ที่ผู้เล่นต้องวางแผนจัดลำดับการ์ดคำสั่ง
(Action Sequence Timeline) เพื่อพาตัวละครวิ่งผ่านสิ่งกีดขวาง หลบหนีออกจากพื้นที่อันตราย 
3 โซนหลัก (โรงเรียน, เมืองร้าง, ป่าดงดิบ) และแก้ไขปริศนามินิเกมเพื่อเอาชนะ Boss ประจำโซน!


[2. การ์ดคำสั่งและการควบคุม (ACTION CARDS)]
------------------------------------------------------------------------------
ผู้เล่นสามารถลากการ์ดคำสั่งลงใน Timeline หรือคลิกเลือกการ์ดคำสั่งดังนี้:

1. WAIT (รอเวลา ⏱️)
   - สั่งให้ตัวละครหยุดรอ 1 จังหวะ
   - เหมาะสำหรับรอให้ไอความร้อน หรือกับดักไฟสลับสถานะเปิด-ปิด

2. JUMP (กระโดด 🏃‍♂️💨)
   - สั่งให้ตัวละครกระโดดสูง
   - ใช้ข้ามสิ่งกีดขวางระดับต่ำ หรือเหว void (Pit Gap)

3. SLIDE (สไลด์หมอบ 🏄)
   - สั่งให้ตัวละครสไลด์หมอบต่ำไปกับพื้น
   - ใช้ลอดผ่านท่อไอร้อน หรือสิ่งกีดขวางระดับหัว

4. DUCK (ย่อตัวหมอบ 🙇)
   - สั่งให้ตัวละครย่อตัวต่ำหลบสิ่งกีดขวางแนวตั้งระดับสูง

5. CLIMB (ปีนป่าย 🧗)
   - สั่งให้ตัวละครปีนป่ายข้ามสิ่งกีดขวางแนวกำแพงสูง


[3. กติกาและการรันลำดับคำสั่ง (GAMEPLAY RULES)]
------------------------------------------------------------------------------
- Timeline Budget: แต่ละด่านจะมีจำนวนการ์ดคำสั่งที่แนะนำ (เช่น 0/2 CARDS)
- การรันคำสั่ง: เมื่อจัดคำสั่งเรียบร้อยแล้ว กดปุ่ม [ RUN SEQUENCE AUTOMATICALLY > ]
  เพื่อให้ตัวละครเริ่มวิ่งปฏิบัติตามลำดับการ์ดที่วางไว้
- การแก้ไข Timeline:
  * กด [ CLEAR TIMELINE ] เพื่อลบการ์ดทั้งหมดใน Timeline
  * กด [ RESET PLAYER ] เพื่อรีเซ็ตตำแหน่งตัวละครกลับมาจุดเริ่มต้น


[4. ระบบคะแนนดาว (STAR RATING SYSTEM)]
------------------------------------------------------------------------------
⭐ ⭐ ⭐ (3 ดาว): ผ่านด่านโดยใช้จำนวนการ์ดคำสั่งไม่เกินโควตา Timeline Budget
⭐ ⭐   (2 ดาว): ผ่านด่านแต่ใช้การ์ดคำสั่งเกินโควตาที่กำหนด
⭐     (1 ดาว): ผ่านด่านขีดสุด
* การปลดล็อกโซนถัดไป (City / Forest) จำเป็นต้องสะสมจำนวนดาวรวมตามที่กำหนด!


[5. ด่านบอสและมินิเกม (BOSS STAGES & MINIGAMES)]
------------------------------------------------------------------------------
เมื่อผู้เล่นเดินทางถึงด่านสุดท้ายของแต่ละโซน จะต้องเผชิญหน้ากับด่านบอส:

• Level 8 (Boss 1 - School Sector): 
  - มินิเกม 8-Puzzle (Sliding Tile) 
  - สไลด์แผ่นตัวเลข 1-8 ให้เรียงลำดับอย่างถูกต้องเพื่อเปิดประตูหนี

• Level 15 (Boss 2 - City Sector): 
  - มินิเกม Pipe Mania (Pipe Connection) 
  - หมุนท่อน้ำและสายไฟเชื่อมต่อวงจรพลังงานให้สมบูรณ์

• Level 22 (Boss 3 - Forest Sector): 
  - มินิเกม Tower of Hanoi (หอคอยฮานอย) 
  - ย้ายแผ่นจานดิสก์ทั้งหมดไปยังเสาเป้าหมายโดยห้ามวางแผ่นใหญ่ทับแผ่นเล็ก


[6. ระบบเนื้อเรื่องและคัทซีน (CUTSCENE & STORY)]
------------------------------------------------------------------------------
- มีคัทซีนเนื้อเรื่องเล่าเหตุการณ์ก่อนเข้าด่านบอสและหลังชนะบอส
- ข้อความคัทซีนแสดงผลแบบตัวอักษรพิมพ์ทีละคำ (Typewriter Dialogue)
- การเลื่อนคัทซีน: คลิกตรงไหนก็ได้บนหน้าจอเพื่ออ่านข้อความถัดไป
- การข้ามคัทซีน: กดปุ่ม [ SKIP CINEMATIC > ] ที่มุมบนขวาเพื่อข้ามคัทซีนทันที


[7. ระบบบันทึกเกมและเงื่อนไข 2 วัน (SAVE SYSTEM & 2-DAY EXPIRATION)]
------------------------------------------------------------------------------
- เกมจะทำการเซฟความคืบหน้าด่านล่าสุดและดาวที่สะสมไว้อัตโนมัติทุกครั้งที่ผ่านด่าน
- เงื่อนไขเวลา 2 วัน (172,800 วินาที):
  * หากเปิดเกมเข้ามาเล่นภายใน 2 วัน ปุ่ม [ CONTINUE ESCAPE ] หน้าเมนูจะทำงาน 
    ให้กดเล่นต่อจากด่านล่าสุดได้ทันที
  * หากไม่ได้เล่นเกิน 2 วัน หรือไม่มีไฟล์เซฟ ระบบจะลบเซฟเก่าทิ้งอัตโนมัติ 
    และแสดงเฉพาะปุ่ม [ NEW GAME ] เพื่อเริ่มความท้าทายใหม่


[8. การตั้งค่าและปุ่มลัด (SETTINGS & SHORTCUTS)]
------------------------------------------------------------------------------
- กดปุ่ม [ ⚙️ SETTINGS ] ในเกมหรือหน้าเมนู เพื่อปรับระดับเสียงเพลง (BGM) 
  และเสียงเอฟเฟกต์ (SFX)
- กดปุ่ม [ < BACK TO SELECTION ] เพื่อกลับไปยังหน้าเลือกด่าน (Level Select)
